ATMS Mobile Demo

System Requirements

  • Apple iOS 5.1 or above. Supported iOS devices are iPad 2/3/4/Mini/Air.
  • Google Android 4.1 or above. All Android tablet devices are supported.

Download Instructions

  1. Open a web browser session on your mobile device. On the iPad this would typically be Safari and on an Android device this would typically be the app labeled "Internet".
  2. Navigate to the URL https://members.aqtsolutions.com/demo/mobile (this page).
  3. Download the ATMS Mobile Workspace for your appropriate device.
  4. After the app has completed installation, you will have a preconfigured ATMS Mobile Workspace with an ATMS Mobile application within it pre-configured to run against an AQT Demo database.
  5. You can run ATMS Mobile by opening the "ATMS Mobile Workspace" and clicking on "ATMS Mobile" from within.

Congratulations! You now have a working ATMS Mobile demo installed on your device. You may use this demo to evaluate the application. Changes to this demo database are reverted daily.

Using the ATMS Mobile Demo

Our ATMS Mobile Demo download is configured to run against a demo database hosted on AQT servers. This demo database is configured to offer some broad overviews of ATMS Mobile to be used to evaluate ATMS Mobile functionality. Below is some basic information in order to help you get started.

Login

ATMS Mobile is designed to be used by instructors and we offer two demo instructor logins:

Navigation

ATMS Mobile has two navigation areas:

  • Navigation Bar
  • Action Bar

The navigation bar is located in the header (top of the screen) and is used to navigate between different screens/areas of ATMS Mobile. When you are at the home screen (Event List), it also contains an "Exit" button to close the app.

The action bar is located in the footer (bottom of the screen) and is used to perform various actions from within the selected area. These actions change according to where you are in ATMS Mobile.

Grading

Grading is available from the "Event List" screen of ATMS Mobile. To grade an example event, simply select an event and hit "Grade" in the action bar.

Ungrading functionality is also offered in ATMS Mobile. To ungrade an event, you will have to select an event with a "Completed" status and then hit "Ungrade" in the action bar. If you are having trouble ungrading, make sure your search criteria is showing events with the "Complete status". Use the "Search" button in the "Event List" navigation bar to view your currently selected search criteria.

Disconnected Use

One of the key features of ATMS Mobile is that it allows end-users to use the application while disconnected (offline). ATMS Mobile allows you to download events from the "Event List" in order to allow grading while offline.

To download an event, select an event from the "Event List" and hit "Download" in the action bar.

This event will now be available in the "Downloaded Event List" which can be accessed by hitting the icon on the left side of the action bar. Events in this list will also be available for grading while the mobile device is disconnected from the internet.

Once you are reconnected, you can upload downloaded events by selecting them in the "Downloaded Event List" and hitting "Upload" in the action bar.

Help

Help using ATMS Mobile can be found by clicking on the icon at the bottom right corner of each page in the app. Also, you can click here to download the ATMS Mobile User's Guide in PDF format.

Note: The built-in help documentation for ATMS Mobile is currently in development.
