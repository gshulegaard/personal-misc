Printing

This ATMS web client supports the printing of reports and other documents stored within the ATMS training management system. In order to print, just follow these steps.

  1. Select the "Print..." option from the RMB menu.

    Note: This option is available within ATMS for various printable resources.

  2. A helpful side-menu will appear on the left side of your browser window with your desired printed resource in it.

  3. Simply click "Download" for your desired resource and you will be able to download a printable .pdf. Your download process will differ depending on your browser, but you should choose to either "Save" or "Open" the downloaded file when applicable.

    For example, with Google Chrome a donwload bar will appear at the bottom of your window.

  4. Now you can open the download .pdf and print to any printer connected to your local machine.

It is important to note that in order to support remote printing, ATMS converts all printable documents into .pdf files before sending them to the client.

Uploading Files

The portal supports uploading files into ATMS directly from the browser. There are two ways to upload files.

Click and Drag:

  1. Click and drag a file onto the browser window that has your ATMS client running.

  2. A helpful side-menu will appear on the left side of your browser window showing the upload progress for the file you are uploading.

  3. Once the upload is finished, the file will be placed in a drive named "A on ATMS RDP" that is made available to ATMS.

  4. Now you will be able to attach the uploaded file into ATMS where applicable.

    For example, you will be able to select the file in "Attachment Maintenance".

From the side menu:

  1. Open the side menu by clicking on the "down" caret in the upper right icon group.

  2. Use the "Upload Files" button to select a file to upload from your local device.

  3. Once the upload is finished, the file will be placed in a drive named "A on ATMS RDP" that is made available to ATMS.

  4. Now you will be able to attach the uplaoded file into ATMS where applicable.

    For example, you will be able to select the file in "Attachment Maintenance".

Downloading Files

This ATMS web client also supports the downloading of files. In most situations it is preferable to use the printing mechanism, however, downloading is supported.

In the drive "A on ATMS RDP" there is a "Download" folder:

Any file saved in this folder will be automatically downloaded to your web browser.

Example:

You can use the "Save As..." RMB menu option to save a file from ATMS into the "Download" folder:

  1. Select the "Save As..." option from the RMB menu.

    Note: This option is available within ATMS for various resources that can be saved.

  2. Make sure to save the resource into the "Download" directory on the "A on ATMS RDP" file system.

  3. A helpful side-menu will appear at the top of your browser window with your saved file in the "File Transfer" area...

  4. Simply click "Download" for your desired resource and you will be able to download the file. Your download process will differ depending on your browser, but you should choose to "Save" the downloaded file when applicable.

    For example, with Google Chrome a download bar will appear at the bottom of your window.

  5. Now you can open the downloaded file from your local device.

Touch Screens

The ATMS web RDP solution has robust touch screen support including click-and-drag and right-click commands. This guide describes how to use these commands from a touch-enabled device.

Touch to click

By default, the ATMS web portal uses "touch-to-click" commands from touch-enabled devices. This means that you may simply touch anywhere to simulate a "click" wherever your finger is located.

This also means that double-tapping simulates a double-click.

Right click (RMB)

To perform a right-click (RMB), touch and hold the position on the screen you wish to perform the right-click.

For example, within ATMS a RMB opens a menu with actions for the specific window you are on:

Double tap (and HOLD) to drag

In order to drag the mouse in a clicked mode, like dragging a scroll bar or moving a window, simply double-tap and hold.

Just like for double-clicking, double-tap, but instead of lifting your finger on the second tap, hold it down for a moment and you will be able to drag your mouse in a clicked mode.

For example, double tapping and holding a scroll bar will allow you to slide it up and down, scrolling through a window's contents as if you were clicking and dragging with a mouse.

Change Password

You can access the "Change Password" form by clicking on the "Settings" icon in menu in the upper-right corner of the browswer window.

This will open a modal dialog with a simple form in it:

Simply enter your new desired password twice and click "Submit". The "Submit" button will be disabled until you fill out both fields. If you do not enter the same password in both fields, an error message will be displayed.

If successful, a success message will display. You will be able to start using your new password immediately (on next login).

Forgot Password

In addition to changing passwords, this web application supports the recovery of forgotten passwords.

You can find the "Forgot Password" button on the login screen:

This button will not be available until you enter a username.

If you click on "Forgot Password", ATMS will attempt to send a password recovery e-mail to the specified user. In order for an e-mail to be sent, you have to:

  1. Enter a valid user account.
  2. Configure your user account with an associated e-mail address. If your user account doesn't have an e-mail address defined, ATMS will not know where to send the recovery e-mail and you will not be able to sue the password recovery feature.
